Fitur Dari Website Tugas Ini

Login

Login dengan memasukkan data yang telah didaftarkan. Data akan diperiksa melalui metode POST dan dicocokkan dengan database.

Login

Register

Form pendaftaran untuk menyimpan data ke database. Data yang dimasukkan meliputi username, email, dan password.

Register

Edit Data Pengguna

Fitur ini memungkinkan admin untuk mengedit data pengguna yang terdaftar di website ini.

Role

Fitur untuk menetapkan role pengguna (admin/user). Role dapat diubah oleh admin.

Hapus Pengguna

Admin dapat menghapus pengguna yang terdaftar menggunakan fitur ini.

Logout

Fitur untuk keluar dari website dan kembali ke halaman utama.

Model Database

Database bernama `flask_login_last` dengan atribut: id, username, email, password_hash, dan role.
